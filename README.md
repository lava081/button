# button
存放Lain-plugin的QQBot适配器模板按钮

## 1.安装插件

在`Miao-Yunzai`根目录执行，任选其一

Gitee：
```
cd ./plugins/Lain-plugin/plugins/button
git init
git remote add origin https://gitee.com/lava081/button
git checkout master
git pull
```

Github：
```
cd ./plugins/Lain-plugin/plugins/button
git init
git remote add origin https://github.com/lava081/button
git checkout master
git pull
```
